# 社区医院管理系统 java490

## 项目概述

社区医院管理系统是一个基于Springboot+Vue的前后端分离项目，采用MVC三层架构，以Spring、SpringBoot、MyBatis、Vue和Maven技术栈实现。本项目使用MySQL数据库作为数据存储，构建了一个B/S结构的Web系统，界面简洁，操作便捷。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 构建工具：Maven
- 架构：MVC

## 功能模块

- 用户管理
- 病患信息管理
- 医生与排班管理
- 药品库存管理
- 费用结算
- 系统设置

## 系统角色

- 系统管理员
- 医生
- 药剂师
- 挂号收费员

## 运行环境

- Java 8 或更高版本
- MySQL 5.7 或更高版本
- Maven 3.5 或更高版本

## 部署步骤

1. 搭建Java运行环境。
2. 安装MySQL数据库，创建对应的数据表。
3. 使用Maven编译项目。
4. 部署前端Vue项目。
5. 部署后端Springboot项目。

## 目录结构

```

│   ├── node_modules

│   └── ...
└── 后端项目Springboot
    ├── src

    │   │   ├── java
    │   │   ├── resources
    │   │   └── ...
    └── ...
```

## 核心亮点

- 前后端分离设计，便于管理和维护。
- 使用Springboot快速开发和部署。
- 采用Vue，实现了响应式和友好的用户界面。
- MVC架构模式，确保系统的高内聚和低耦合。
